# Recipe Search

This is a recipe search app made with React JS.

### Installing

Install dependencies

```
npm install
```

Start the server

```
npm start    Served on localhost:3000
```

Build for production

```
npm run build
```

## Built With

- [React js](https://reactjs.org/)
- [Food2Fork API](https://www.food2fork.com/about/api)
